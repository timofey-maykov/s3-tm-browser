# S3 TM Browser

English version: [README.en.md](README.en.md)

S3 TM Browser это кроссплатформенное настольное приложение для работы с объектными хранилищами S3.
Оно подходит для AWS S3 и S3-совместимых сервисов.
Поддерживаются macOS, Windows и Linux.

## Что умеет приложение

- Подключение к нескольким аккаунтам, endpoint и регионам
- Поддержка AWS S3 и S3-совместимых хранилищ
- Просмотр бакетов, префиксов и объектов в табличном виде
- Поиск и фильтрация объектов внутри бакета
- Создание и удаление бакетов
- Загрузка файлов и папок
- Скачивание файлов, папок и целых бакетов
- Drag and drop для загрузки
- Копирование и перемещение объектов между бакетами и профилями
- Переименование файлов и папок
- Удаление одиночных и массово выбранных объектов
- Очередь передач с паузой, продолжением и отменой
- Отображение статуса, прогресса и ошибок операций
- Работа с multipart загрузками
- Просмотр и отмена незавершенных multipart загрузок
- Работа с версиями объектов
- Просмотр, скачивание, восстановление и удаление версий
- Просмотр метаданных и тегов объектов
- Редактирование пользовательских метаданных и тегов
- Просмотр ACL объектов и бакетов
- Массовые операции ACL для выбранных объектов
- Генерация presigned ссылок
- Просмотр статических ссылок virtual-host и path-style
- Превью текстовых и поддерживаемых файлов
- Просмотр и управление настройками бакета
- Bucket Policy, CORS, Lifecycle, Encryption, Logging, Versioning
- Public Access Block и связанные параметры доступа
- Двухпанельный режим Коммандер
- Копирование и синхронизация между панелями
- Сравнение содержимого между двумя S3 панелями
- Горячие клавиши для частых операций
- Настройки приложения для очереди, ретраев, скорости и логов
- Диагностический снимок для передачи в поддержку
- Импорт подключений из S3 Browser на Windows

## Для кого

Приложение подходит для администраторов, разработчиков и команд,
которым нужен понятный десктопный инструмент для ежедневной работы с S3.
