# S3 TM Browser

English version: [README.en.md](README.en.md)  
中文版本: [README.zh.md](README.zh.md)

S3 TM Browser это кроссплатформенное настольное приложение для работы с объектными хранилищами S3.
Оно подходит для AWS S3 и S3-совместимых сервисов.
Поддерживаются macOS, Windows и Linux.

## Что умеет приложение

- Подключение к нескольким аккаунтам, endpoint и регионам
- Поддержка AWS S3 и S3-совместимых хранилищ
- Просмотр бакетов, префиксов и объектов в табличном виде; индикатор прогресса при многостраничном листинге с отменой; экспорт текущего списка в JSON и CSV из тулбара
- Поиск и фильтрация объектов внутри бакета (включая глубокий поиск по ключам под текущим префиксом, с лимитами и отменой)
- Оценка суммарного размера текущего префикса или выбранных объектов и префиксов (тулбар обзора)
- Создание и удаление бакетов
- Загрузка файлов и папок
- Скачивание файлов, папок и целых бакетов
- Параллельное скачивание крупных объектов по HTTP Range: в параметрах multipart задаются размер части и число одновременных Range (до 128, с ограничением суммарного объёма буферов в полёте); в очереди передач для таких задач показывается прогресс по сегментам, как при multipart загрузке
- Drag and drop для загрузки
- Копирование и перемещение объектов между бакетами и профилями; мастер миграции (проверка доступа, оценка объёма префикса, постановка копирования в очередь)
- Переименование файлов и папок
- Удаление одиночных и массово выбранных объектов
- Очередь передач с паузой, продолжением и отменой; оценка оставшегося времени и строка скорости по ходу задачи; детали задачи с прогресс-баром и разбором источника и назначения; визуальный акцент на полосках прогресса для активных задач
- Отображение статуса, прогресса и ошибок операций
- Работа с multipart загрузками
- Просмотр и отмена незавершенных multipart загрузок
- Работа с версиями объектов
- Просмотр, скачивание, восстановление и удаление версий
- Просмотр метаданных и тегов объектов
- Редактирование пользовательских метаданных и тегов
- Просмотр ACL объектов и бакетов
- Массовые операции ACL для выбранных объектов
- Управление Object Lock бакета, retention и legal hold для объектов
- Серверное шифрование объектов (SSE-S3, SSE-KMS, SSE-C): смена режима и снятие шифрования; ключ SSE-C в keyring профиля или однократная подстановка в диалоге; пакетная расшифровка выбранных объектов и префиксов в отдельном окне с проверкой ключа base64
- Генерация presigned ссылок
- Просмотр статических ссылок virtual-host и path-style
- Превью текстовых и поддерживаемых файлов
- Просмотр и управление настройками бакета
- Bucket Policy, CORS, Lifecycle, Encryption, Logging, Versioning, Object Lock
- Public Access Block и связанные параметры доступа
- Двухпанельный режим Коммандер
- Копирование и синхронизация между панелями
- Сравнение содержимого между двумя S3 панелями
- Сниппеты rclone и примеры AWS CLI для текущего префикса в тулбаре обзора (без секретов)
- Контекстное меню тулбара обзора: видимость кнопок и колонок таблицы, подгонка ширины колонок по содержимому, сброс вида
- Импорт ключей из отчёта S3 Inventory: CSV, архив `csv.gz`, `manifest.json`; постановка копирования списка ключей в очередь
- Горячие клавиши для частых операций
- Глобальные и профильные настройки прокси (HTTP, env auto); пресет вендора S3-совместимого endpoint для подсказок при ошибках
- Настройки приложения для очереди, ретраев, ограничения скорости передач, multipart (размер частей загрузки и скачивания, число параллельных частей загрузки и параллельных Range при скачивании, до 128 с учётом лимита буферов в полёте), целостности при скачивании (sha256 и crc32c из метаданных и ответа HEAD), выбора контрольной суммы для небольших загрузок и логов
- Диагностический снимок для передачи в поддержку
- Интерфейс на русском, английском и китайском языках; встроенная справка в Настройках (оглавление и поиск по разделам)
- Командная строка `s3tm-cli` для операций из терминала
- Импорт подключений из S3 Browser на Windows

## Для кого

Приложение подходит для администраторов, разработчиков и команд,
которым нужен понятный десктопный инструмент для ежедневной работы с S3.
